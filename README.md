# StreamController Device Store

Custom store for StreamController device plugins.

## Available Device Plugins

| Plugin | Description | Author |
|--------|-------------|--------|
| [Mirabox Devices](https://github.com/crusardri/MiraboxStreamController) | Support for Mirabox N3/N4 and compatible devices | crusardri |

## How to Use

Add this store to StreamController:

1. Open StreamController → Settings → Store
2. Enable "Custom Stores"
3. Add a new entry:
   - **URL:** `https://github.com/crusardri/StreamController-DeviceStore`
   - **Branch:** `main`
4. Reopen the Store to see available device plugins

## Contributing

To add your device plugin to this store, submit a Pull Request with your plugin added to `Devices.json`.
